# Evaluation Cases

After completing an audit, use `EVAL.md` (at repo root) to grade the report.

## Adding a New Case

1. Copy `cases/template.md`
2. Fill in the CE details and scores
3. Name it `cases/<ce-slug>-<date>.md`

## Eval Process

1. Read SKILL.md, DIAGNOSTICS.md (know what instructions existed)
2. Read the completed report
3. Score each of 7 themes (1-5)
4. Classify every gap with a failure mode tag
5. Identify top 2-3 improvements

See EVAL.md for full rubric and scoring thresholds.
